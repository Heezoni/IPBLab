Scentone:

Scentone is color cube diffuser. Each blocks have different scent and color.Using the image that be associated with the scent, it matches to the color, so that the scent and color harmonize well. With two blocks, the scent can be physically mixed and separated to create the combination of different scents. Scentone can be used as an interior object in the home and offices.
